PaintStream
===========

Allows you to paint with your friends

<p>Installation Guide</p>
<ol>
  <li>install mongodb</li>
  <li>install node</li>
  <li>clone repo</li>
  <li>npm install</li>
  <li>npm install faye</li>
</ol>

<p>Run Guide</p>
<ol>
  <li>mongod (run mongodb)</li>
  <li>node app.js (run app)</li>
</ol>
